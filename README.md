## Local Build

1. Clone the repo

```
git clone https://github.com/PoromKamal/Porom.git
```

2. Install Dependencies

```
cd %PROJECT_ROOT%
npm i
```

3. Run the project
```
# In the project root
npm run dev
```

4. Open the website on localhost:3000
